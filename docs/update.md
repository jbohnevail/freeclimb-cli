`freeclimb update`
==================

Check for newer versions of the FreeClimb CLI.

Queries the npm registry and compares against the installed version.
If an update is available, shows installation instructions.

* [`freeclimb update`](#freeclimb-update)

## `freeclimb update`

Check for newer versions of the FreeClimb CLI.

```
USAGE
  $ freeclimb update [--json] [-h]

FLAGS
  -h, --help  Show CLI help.
      --json  Output as JSON (for scripting/agents)

DESCRIPTION
  Check for newer versions of the FreeClimb CLI.

  Queries the npm registry and compares against the installed version.
  If an update is available, shows installation instructions.
```
